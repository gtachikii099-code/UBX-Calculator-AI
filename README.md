# UBX Calculator AI v7 — Node.js Only

- No Express
- No npm install
- Login and Signup routes fixed and tested
- Persistent sessions
- OpenRouter AI
- Markdown formatting and chat history
- Calculator, Premium, Settings and Admin

## Start
1. Run SETUP-OPENROUTER.bat and add the key.
2. Run START-APP.bat.
3. Open http://localhost:5512
